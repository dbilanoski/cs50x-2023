## HTML

Hypertext markup language, structure and semantcis of web page - basicaly a document.

Has tags and attributes.

```html
<!DOCTYPE html>

<html lang="en">
  <head>
    </title>
      Hello, title
    </title>
  <body>
    <h1>Hello, body</h1>
  </body>
</html>
```
